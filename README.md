# Week2-Task1-EDA

## Project Overview

This project demonstrates Data Collection, Data Cleaning, and Exploratory Data Analysis (EDA) using the Titanic dataset. The objective is to preprocess the data, handle missing values, explore patterns, and visualize important insights using Python.

---

## Dataset

- Dataset: Titanic Dataset
- File: `raw_dataset.csv`

## Steps Performed

1. Loaded the Titanic dataset.
2. Explored the dataset structure.
3. Checked missing values.
4. Cleaned the dataset.
5. Saved the cleaned dataset.
6. Performed Exploratory Data Analysis (EDA).
7. Created visualizations for better understanding.
